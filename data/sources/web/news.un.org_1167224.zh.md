# 二战后首次：战争海域被困船员面临"史无前例"危机

**作者**

**丹尼尔·迪金森，伦敦**

随着中东战事持续，约2万名船员被困在霍尔木兹海峡的船只上，这种情况被描述为二战后时代前所未有的局面。

这些船员分布在约2000艘船舶上，包括油气运输船、散货船、货轮以及六艘旅游邮轮。由于中东持续战事，这些船只被困在波斯湾，无法通过狭窄的霍尔木兹海峡。

伊朗控制着海峡北侧，并宣称仅允许"非敌对"船只通过。战前该水道日均通行150艘船只，如今仅剩四五艘。周一，两艘悬挂中国国旗的货轮耗时4-6小时成功穿越海峡，驶入阿曼湾及战区外的安全水域。

**船舶遇袭事件**

据联合国国际海事组织（IMO）伦敦总部数据，过去一个月冲突期间已发生19起海峡船只遇袭事件。自美以开始轰炸伊朗引发海湾地区报复性打击以来，已造成10名船员死亡、8人受伤。周二，一艘满载油轮在迪拜海岸附近疑似遭武装无人机袭击。

目前尚不清楚这19艘船只为何成为特定攻击目标。随着外交解决危机的努力加强，过去一周袭击事件似有减少。

**船员安全保障**

作为联合国专门负责提升国际航运安全的机构，IMO正全力确保2万名船员的撤离与安全。"现代航运史上从未出现过如此大规模的船员被困事件，"IMO海事安全司司长达米安·雪瓦利耶表示，"我们呼吁冲突各方停止升级袭击，以便船员能安全撤离。"

"他们已在交战区工作整整一个月，"雪瓦利耶指出，"这种极端恐惧状态下的心理压力难以想象。"代表船员利益的国际运输工人联合会表示，已收到逾千封被困船员的求助邮件，反映船上恶劣条件并请求遣返回国。

"理论上可通过轮换船员解决问题，但船运公司需要找到自愿接替者，"雪瓦利耶说，"最根本的解决方案是让船只安全通过霍尔木兹海峡——但这需要敌对行动首先停止。"

**安全通道谈判**

目前波斯湾的2000艘船只正通过沙特和阿曼的补给公司获取食物、饮水和燃料。沙特当局与IMO合作向航运业提供了补给公司联络方式。由于港口也非绝对安全，各船正根据船东协议在海湾机动寻找安全泊位。

雪瓦利耶透露，IMO已要求伊朗澄清"敌对船只"的界定标准，以明确哪些船舶通过海峡时会面临攻击威胁。

**国际航道体系**

该海峡对全球经济至关重要，全球约20%的油气供应经此运输。1968年IMO与该地区国家协议通过了双向分道通航制，规划出紧贴阿曼南部的最安全航线。但近期少数通行船只选择靠近伊朗的北部航线，据称是为便于伊朗当局密切监控。

**船员未来何去何从？**

IMO短期目标是确保被困船员安全，但更长远关注航运业人才储备。"如果因当前这类冲突导致船员安全感丧失，将难以吸引新生代满足日益增长的航运需求，"雪瓦利耶强调，"没有船员，世界经济所依赖的全球贸易就无法运转。"
