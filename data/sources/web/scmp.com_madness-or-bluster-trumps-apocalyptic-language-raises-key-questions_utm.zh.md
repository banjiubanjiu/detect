# 疯狂还是虚张声势？特朗普的末日言论引发关键质疑

最后一刻的停火阻止了"抹除伊朗文明"的威胁，但加剧了人们对特朗普谈判风格和精神状态的质疑

![美国总统特朗普周一参加白宫年度复活节滚彩蛋活动，尽管与伊朗的战争持续，他仍保持公开行程。照片：路透社](https://cdn.i-scmp.com/sites/default/files/styles/1020x680/public/d8/images/canvas/2026/04/08/ab1d0edc-7d0d-441f-a48d-57b63cf75eb4_57f741b8.jpg?itok=peO6F13Y&v=1775629223)

这位曾渴望获得诺贝尔和平奖、一度沉醉于解决冲突形象的总统，在竭力寻求伊朗战争解决方案时，转而使用了毁灭性语言。

美国总统特朗普周二就伊朗战争发出的最新威胁达到新极端，他警告称"整个文明将在今晚消亡，永远无法恢复"，除非伊朗达成包括重新开放霍尔木兹海峡关键航道的协议。

这位共和党总统的言论引发全球震惊，美国国内民主党人、部分已与特朗普决裂的"让美国再次伟大"支持者、以及首位美籍教皇都予以谴责。一些共和党同僚暗示他的言论只是谈判策略。

此前数日他接连威胁将"把伊朗炸回史前时代！！！"并表示要炸毁桥梁和民用发电厂，军事法专家指出这可能构成战争罪。复活节清晨，他在社交媒体账号发文："他X的打开海峡，你们这群疯子，否则就等着下地狱吧。"

就在晚8点最后期限（香港时间周三早8点）前一小时，特朗普宣布撤回大规模打击威胁，条件是伊朗同意两周停火并重新开放霍尔木兹海峡。伊朗最高国家安全委员会表示已接受两周停战协议。
