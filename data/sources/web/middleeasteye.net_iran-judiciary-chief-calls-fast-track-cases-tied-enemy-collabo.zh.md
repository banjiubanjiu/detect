# 伊朗司法部长要求加速审理"通敌合作"案件

2026年4月7日 07:20 英国夏令时

![](/sites/default/files/styles/max_2600x2600/public/2026-04/000_A6XH2EW.jpg.jpg?itok=az6vTcg6)

伊朗司法部长呼吁对涉嫌协助境外敌对势力者加快起诉程序。在美以持续空袭的背景下，该国逮捕行动仍在继续。

据米赞通讯社报道，古拉姆侯赛因·穆赫辛尼·埃杰伊周二发表讲话时表示，应加速审理涉嫌协助敌对行为者的案件。他敦促司法机构加快判决进程，包括执行死刑和没收财产等裁决。

自上月美以发动联合打击以来，伊朗安全部队已以涉嫌通敌为由拘押数百人。
