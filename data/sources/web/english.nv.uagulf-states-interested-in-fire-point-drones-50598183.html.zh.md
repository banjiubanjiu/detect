# 海湾国家有意采购"火力点"无人机 公司称正等待出口许可

2026年4月7日 00:09

![火力点公司FP-2攻击型无人机（照片：Serhii Okunev/NV）](https://static.nv.ua/shared/system/Article/posters/003/270/044/original/5135ab40a8902e203e617f450cda90e9.png?q=85&stamp=20260407012907&w=900&f=webp)

火力点公司FP-2攻击型无人机（照片：Serhii Okunev/NV）

乌克兰国防承包商"火力点"公司联合创始人兼总设计师丹尼斯·什蒂列尔曼4月6日[向路透社透露](https://www.reuters.com/business/aerospace-defense/ukraine-missile-maker-targets-game-changer-air-defence-system-by-2027-2026-04-06/)，多个海湾国家正寻求[采购](https://english.nv.ua/nation/us-gulf-allies-urged-to-study-ukraine-s-drone-war-lessons-50588454.html)该公司生产的作战无人机，但仍在等待政府发放出口许可证。

什蒂列尔曼表示，公司具备每月出口2500架远程攻击无人机的产能。路透社报道称："什蒂列尔曼还表示，火力点公司正在等待政府批准中东某集团的投资，该投资对公司估值达25亿美元，将为包括低轨道卫星发射在内的新业务机会打开大门。"

他提到"火烈鸟"巡航导弹的出口因监管障碍更为复杂。火力点公司表示，目前每天生产数百架远程攻击无人机，单机成本约5万欧元，同时制造单价约60万欧元的"火烈鸟"导弹。

什蒂列尔曼承认导弹生产存在一些"瓶颈"，特别是发动机制造环节。他表示，待10月份新型自研发动机开始批量生产，以及丹麦固体火箭燃料工厂今年晚些时候投产后，公司将提升"火烈鸟"导弹产量。

什蒂列尔曼还告诉路透社，火力点公司计划在2027年前开发自己的防空系统，作为美国"爱国者"系统的低成本替代方案。
