# 局势进展伊朗将在与特朗普达成的两周停火协议下重新开放霍尔木兹海峡

华盛顿收到"伊朗提出的10点建议"，美国总统称其可作为"可行的谈判基础"

![美国总统唐纳德·特朗普向记者发表讲话，警告在伊朗接受和平协议的最后期限前可能出现的局势升级。照片：美联社](https://cdn.i-scmp.com/sites/default/files/styles/1020x680/public/d8/images/canvas/2026/04/08/ced9e893-eb3c-4e14-b2af-6de5cf25feb7_5bc42f68.jpg?itok=gwsk7Llb&v=1775597409)

伊朗将在巴基斯坦斡旋下与美国达成停火协议，重新开放霍尔木兹海峡，双方还同意于周五在伊斯兰堡启动和平协议谈判。

美国总统唐纳德·特朗普周二表示，美国将暂停对伊朗的轰炸行动两周，称这是巴基斯坦调解方为创造外交空间而提出的"双向停火"要求。

特朗普在社交媒体发文中表示，军事行动的暂停"取决于伊朗伊斯兰共和国同意霍尔木兹海峡的完全、立即和安全开放"。

他补充说，华盛顿已收到"伊朗提出的10点建议"，并认为其可作为"可行的谈判基础"。

特朗普表示："美国和伊朗之间已就过去几乎所有争议点达成一致，但两周时间将有助于协议的最终确定和完善。"

伊朗外交部长阿巴斯·阿拉格希在德黑兰表示："如果针对伊朗的攻击停止，我们强大的武装力量将中止防御行动。"

阿拉格希补充说："在两周时间内，霍尔木兹海峡将恢复安全通航。"
