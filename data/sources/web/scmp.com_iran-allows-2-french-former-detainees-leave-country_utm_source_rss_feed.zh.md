# 伊朗允许两名法国前被拘留者离境

法国长期争取的离境许可，标志着伊朗战争背景下德黑兰对各国采取区别对待策略

![法国国民议会外展示的塞西尔·科勒（左）与其伴侣雅克·帕里肖像及标语牌，上书"释放被伊朗任意拘禁两年多、遭受恶劣待遇的塞西尔·科勒与雅克·帕里"。摄于2025年。（法新社图片）](https://cdn.i-scmp.com/sites/default/files/styles/1020x680/public/d8/images/canvas/2026/04/07/7ecc4b3d-52f3-4b97-9632-19974432a8a3_701b121a.jpg?itok=2CiH8u6N&v=1775572255)

法国总统马克龙周二宣布，伊朗已允许两名法国前被拘留者塞西尔·科勒与雅克·帕里离境。两人自监狱获释后一直滞留在法国驻伊外交机构内。

"经过三年半的伊朗拘押，塞西尔·科勒和雅克·帕里现已获释，正在返回法国领土的途中，"马克龙在社交媒体发文称。

法国长期争取的离境许可，标志着在当前伊朗战争背景下，德黑兰政权正采取区别对待各国的策略——对部分国家示好，同时将另一些国家视为敌手。

马克龙始终与这场冲突保持距离，强调法国未提前获知美以联合打击计划，且反对战争爆发。

![法国国民议会议员们获悉塞西尔·科勒与雅克·帕里结束三年半拘禁即将返法时鼓掌致意。摄于周二巴黎国民议会会议期间。（法新社图片）](https://img.i-scmp.com/cdn-cgi/image/fit=contain,width=1024,format=auto/sites/default/files/d8/images/canvas/2026/04/07/51f9db12-02e0-4e98-acfd-708ce220c640_69875011.jpg)

马克龙特别感谢阿曼在促成两人获释过程中发挥的调解作用。"这对我们所有人——尤其是他们的家人——而言都是莫大的慰藉，"马克龙写道。
