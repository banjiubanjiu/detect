# 联合国安理会未能通过霍尔木兹海峡相关决议

2026年4月7日 17:08 BST

![](/sites/default/files/styles/max_2600x2600/public/2026-04/iran%20university%20bomb%20reuters.jpg.jpg?itok=YPsB7zM9)

在周二的联合国安理会表决中，中国和俄罗斯否决了由巴林提出的决议草案。该决议旨在鼓励各国协调保护霍尔木兹海峡商业航运的行动。

由15个成员国组成的安理会以11票赞成、2票反对和2票弃权的结果未能通过该决议。
