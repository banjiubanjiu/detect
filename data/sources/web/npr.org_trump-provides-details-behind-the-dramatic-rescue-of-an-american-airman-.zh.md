# 特朗普披露美军飞行员在伊朗境内戏剧性营救行动细节

#### 特朗普披露美军飞行员在伊朗境内戏剧性营救行动细节

特朗普总统详细披露了美军如何紧急营救一架深入伊朗境内被击落战机的两名机组人员的惊险过程。

主持人艾尔莎·常：
特朗普总统今日详细披露了美军如何紧急营救一架深入伊朗境内被击落战机的两名机组人员。目前两名获救人员正在康复中。本台国家安全记者格雷格·迈尔将带来更多细节。你好，格雷格。

格雷格·迈尔：
你好，艾尔莎。

常：
据了解特朗普总统详细讲述了军方如何找到这两名空军军官。请具体说明总统描述的整个行动过程。

迈尔：
总统称这是一次从始至终充满风险的大规模行动。这架F-15战机于周五黎明前被击落，两名机组人员弹射逃生但落点相距甚远。呼号为"DUDE 44 Alpha"的飞行员最先被定位，特朗普表示军方派出21架飞机进入伊朗执行这项极度危险的任务——当时正值白昼，且多数是低速飞行器。营救人员在伊朗领空经历了7小时险境后才成功将其救出。

常：
那第二名机组人员呢？

迈尔：
他的救援过程更为惊险。这名呼号为"DUDE 44 Bravo"的上校是武器系统军官，特朗普称其伤势严重大量失血，但依然...

（档案录音）
特朗普总统：
"他开始向更高海拔爬升，这是为躲避抓捕进行的训练内容。他们必须尽可能远离击落地点，因为敌方会立即向该区域集结。"

迈尔：
虽然上校携带了定位信标，但直到周六早上中情局联系到他时，才发现他藏身于山体裂缝中。

常：
太可怕了。伊朗知道有美军人员失踪，美军如何抢先找到他的？

迈尔：
这正是难点所在。中情局散布假消息称已通过陆路转移获救人员。特朗普称共派出155架飞机进入伊朗领空，这些飞机集中在7个不同区域以迷惑伊朗军方，并对通往真实地点的道路实施轰炸。最终由直升机将其转移至农场临时跑道。由于两架接应飞机陷入湿沙，军方不得不调来轻型飞机，并炸毁了被困飞机。

常：
难以置信。格雷格，美军这些卓越的营救能力是如何发展起来的？

迈尔：
这要追溯到1980年4月营救伊朗美国大使馆人质的失败行动。当时因沙漠沙暴导致一架美军直升机与运输机相撞，造成8名军人丧生。这场灾难促使美军彻底重组特种部队，同年成立了海豹六队等单位。而就在上周末，我们看到这些特种部队在46年后，于同一个国家——伊朗境内成功实施了救援。

常：
最后，特朗普总统是否透露战争可能如何收场？

迈尔：
目前似乎处于关键节点，但总统释放了矛盾信号。他仍坚持伊朗必须在明晚前开放霍尔木兹海峡，否则将面临大规模轰炸；同时又称谈判进展良好。简言之，他同时在谈论战争升级与降级的可能性。

常：
以上是NPR记者格雷格·迈尔的报道。非常感谢。

迈尔：
不客气，艾尔莎。
