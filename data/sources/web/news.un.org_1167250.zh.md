# 国际原子能机构总干事对伊朗核电站再遭袭击报道"深表关切"

有关布什尔核电站附近再次遭到投射物袭击的报道，促使国际原子能机构(IAEA)总干事拉斐尔·格罗西于周六表达了深切忧虑。

伊朗官员向原子能机构通报了这次袭击事件——这是近几周来发生的第四起类似事件。伊朗还告知该机构，现场一名安保人员被投射物碎片击中身亡，厂区一栋建筑受到冲击波和碎片影响。

格罗西先生强调，核电站厂址或附近区域绝不应成为攻击目标，他指出附属建筑可能包含关键安全设备。最新事件后尚未报告辐射水平升高。

格罗西再次呼吁保持最大军事克制以避免核事故风险，并重申在冲突期间遵守原子能机构确保核安全与安保的七项基本原则至关重要（见下文）。

此前3月18日布什尔核电站曾遭袭击，距反应堆约350米的一处建筑被击中损毁。虽未报告反应堆受损或人员伤亡，但该机构警告称核设施附近的任何攻击都可能违反关键安全原则。

本月早些时候，在奥地利维也纳机构总部向原子能机构理事会发表讲话时，格罗西强调了军事升级导致核事故的风险，因为"伊朗及该地区许多其他国家拥有运行中的核电站和核研究反应堆，而这些国家正遭受军事攻击"。

**武装冲突中核安全与安保七项基本原则**

这七项不可或缺的原则由原子能机构总干事于2022年3月提出，旨在应对战区内维持核安全与安保的前所未有挑战。

- 必须保持设施（无论是反应堆、燃料池还是放射性废物储存库）的实体完整性
- 所有安全与安保系统及设备必须始终保持全面运行状态
- 操作人员必须能够履行其安全与安保职责，并拥有免受不当压力影响的决策能力
- 所有核设施必须拥有来自电网的安全场外电力供应
- 必须保持进出场址的物流供应链和运输通道不间断
- 必须建立有效的场内场外辐射监测系统及应急准备与响应措施
- 必须与监管机构及其他相关方保持可靠通讯
