# 停火消息公布前数小时 教皇利奥十四世严厉谴责特朗普

![图片2：身着白色法袍的教皇利奥十四世正在回应一群举着麦克风的记者提问](https://static01.nyt.com/images/2026/04/07/multimedia/07israel-iran-Pope-Trump-vwkq/07israel-iran-Pope-Trump-vwkq-articleLarge.jpg?quality=75&auto=webp&disable=upscale)

教皇利奥十四世周二谴责特朗普总统对伊朗的威胁。图片来源...Guglielmo Mangiapane/Reuters

作者 [Ephrat Livni](https://www.nytimes.com/by/ephrat-livni)

2026年4月7日

首位美国出生的教皇利奥十四世周二罕见地谴责特朗普总统，称威胁要消灭伊朗"整个文明"的做法"完全不可接受"。

他虽未点名，但所指对象不言自明。

"众所周知，今天出现了针对全体伊朗人民的威胁。这完全不可接受，"教皇周二晚间在意大利对记者表示，此时距离特朗普宣布达成两周停火仅数小时。"这当然涉及国际法问题，但更重要的是关乎全体人民福祉的道德问题。"
