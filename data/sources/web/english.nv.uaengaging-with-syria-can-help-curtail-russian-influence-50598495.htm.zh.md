# 与叙利亚接触有助于遏制俄罗斯影响力

2026年4月7日 18:37

![弗拉基米尔·泽连斯基与艾哈迈德·阿尔-沙拉（照片：乌克兰总统新闻服务处/路透社供稿）](https://static.nv.ua/shared/system/Article/posters/003/270/726/original/c07a25c8186fbce49e50a1261b315e37.png?q=85&stamp=20260407214251&w=900&f=webp)

乌克兰军官兼记者尤里·马察尔斯基4月7日向NV电台表示，与[叙利亚](https://english.nv.ua/nation/ukraine-helped-syrian-rebels-with-drones-during-assad-s-fall-report-says-50598429.html)发展安全和经济联系将帮助乌克兰遏制俄罗斯在中东及更广泛地区的影响力。

"叙利亚过去是、现在仍然是一个非常重要的国家，乌克兰在该国拥有重大利益，"马察尔斯基说。"这些利益涉及安全、经济领域，以及削弱俄罗斯在中东乃至更广泛的南半球影响力的努力。"

马察尔斯基指出，俄罗斯试图将自己塑造成在中东和非洲拥有附庸国或至少是高度依赖国家的形象。他提到莫斯科组建的"非洲军团"——一个由瓦格纳集团残余部队和其他雇佣兵组成的官方架构。这些部队目前在马里和中非共和国活动。

马察尔斯基补充说，这些军事组织的运作效能取决于俄罗斯在叙利亚赫梅米姆和塔尔图斯军事基地的持续存在。

"如果乌克兰能与叙利亚当局达成协议，停止或至少严格限制这两个俄罗斯基地的运作，俄罗斯将更难伪装自己并未陷入真正的国际孤立，"他表示。

他警告说，如果俄罗斯通过叙利亚向非洲输送的武器、增援和弹药停止或大幅减少，可能导致非洲军团及其他在非俄军部队崩溃或遭遇重大失败。

**4月5日**，乌克兰总统弗拉基米尔·泽连斯基首次访问大马士革，会晤叙利亚总统艾哈迈德·阿尔-沙拉。泽连斯基表示双方讨论了中东局势及俄罗斯对乌克兰的战争。
