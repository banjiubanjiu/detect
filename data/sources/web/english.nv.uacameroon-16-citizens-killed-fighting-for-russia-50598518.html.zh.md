# 喀麦隆：16名公民在乌克兰为俄罗斯作战时阵亡

2026年4月7日 23:10

![俄罗斯军队（照片：俄罗斯国防部）](https://static.nv.ua/shared/system/Article/posters/003/270/780/original/e6918641c2b13abffef105a5faf80927.png?q=85&stamp=20260408003919&w=900&f=webp)

俄罗斯军队（照片：俄罗斯国防部）

据路透社4月7日[报道](https://www.reuters.com/world/africa/russia-confirms-deaths-16-cameroonians-fighting-ukraine-war-yaounde-says-2026-04-07/)，喀麦隆表示俄罗斯已确认16名[为俄军作战](https://english.nv.ua/russian-war/ukrainian-soldiers-capture-two-russian-mercenaries-from-cameroon-in-donetsk-oblast-video-50525982.html)的喀麦隆公民在乌克兰阵亡。

官方媒体发布的声明称，外交部敦促死者家属与部委官员联系。路透社称该外交文件将死者描述为在所谓"特别军事行动"区域作战的"喀麦隆国籍军事承包商"。

声明和外交照会均未说明这些喀麦隆人如何加入俄军，也未提供关于其阵亡地点、时间或具体情况的详细信息。

报道补充称："俄罗斯驻雅温得大使馆未立即回应路透社的置评请求。"

今年2月，乌克兰表示有超过1700名非洲籍人员在俄方作战，不过分析人士告诉路透社实际数字可能更高。2月21日，乌克兰对外情报局称截至2025年底，已有来自36个非洲国家的1400多名公民为俄罗斯作战。该机构表示招募者主要针对社会弱势青年、教育和就业机会有限者以及有军事经验者，常以安保、司机或建筑工人等民事工作为诱饵。
