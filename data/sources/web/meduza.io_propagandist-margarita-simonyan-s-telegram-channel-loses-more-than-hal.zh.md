# 俄罗斯封杀Telegram后 宣传家西蒙尼扬频道浏览量暴跌过半

据Telegram频道"ChTD"（"证毕"）计算，宣传家玛格丽塔·西蒙尼扬的Telegram频道浏览量在三月下旬较二月下旬下降了52.3%。

ChTD发现，在亲克里姆林宫的Telegram媒体和博主中，只有"克里姆林宫新闻"的跌幅更大，浏览量下降68.1%。由西蒙尼扬领导的俄罗斯官方宣传机构RT的Telegram频道浏览量下降了42%。

ChTD表示，尽管俄罗斯大力推广官方支持的即时通讯应用Max，但仅有少量亲政府Telegram频道的读者迁移至该平台。Max的完整浏览趋势统计数据尚不可得，但根据该应用自身数据显示，西蒙尼扬的帖子在二月下旬平均浏览量约为60万次（订阅者不足13万），到三月下旬已降至约三分之一。"我们可以解释"Telegram频道报道称，西蒙尼扬目前在Telegram拥有53.7万订阅者，在Max上约16.5万。

"我们可以解释"频道透露，RT每年从俄罗斯联邦预算中获得约320-330亿卢布拨款。

独立媒体基本未受影响。ChTD指出，这些媒体的读者长期依赖VPN和其他规避工具。据其报道，同期BBC俄语版浏览量仅下降9.1%，Meduza仅下降3.7%。

三月下旬，独立俄罗斯媒体《新报欧洲》曾评估过Telegram限制措施对亲政府和独立媒体频道的影响。当时该媒体发现，亲政府Telegram频道浏览量下降23%，地区频道下降25%，独立媒体下降10%。
