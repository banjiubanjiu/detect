# 卢克石油公司NORSI炼油厂遭复活节无人机袭击后停产

2026年4月7日 19:02

![NORSI炼油厂 (图片来源: Loris-nn.ru)](https://static.nv.ua/shared/system/Article/posters/003/270/731/original/cd5fe3e0606e48a74add9e89ea5ca01e.png?q=85&stamp=20260407215842&w=900&f=webp)

NORSI炼油厂 (图片来源: Loris-nn.ru)

据路透社4月7日[报道](https://www.reuters.com/business/energy/russias-major-norsi-oil-refinery-halted-operations-after-sunday-drone-attack-2026-04-07/)，乌克兰无人机袭击导致俄罗斯大型[卢克石油炼油厂](https://english.nv.ua/nation/ukraine-strikes-lukoil-refinery-in-russia-50597823.html)停产，预计4月底前难以恢复运营。消息源自俄罗斯石油行业内部人士。

该炼油厂全称为卢克石油-下诺夫哥罗德石油有机合成公司(NORSI)，是俄罗斯第四大炼油企业，也是第二大汽油生产商。路透社称工厂在遇袭后已停止运营，"临时关闭将进一步加剧俄罗斯能源行业的不确定性"，此前该国出口基础设施已多次遭袭。

下诺夫哥罗德州长格列布·尼基京表示，无人机袭击造成炼油厂两处设施受损，并波及一座发电厂和数栋民宅。

圣彼得堡国际商品交易所数据显示，卢克石油公司尚未出售该炼油厂的汽油、柴油或燃料油。卢克石油未回应路透社的置评请求。

报道补充称，NORSI炼油厂年原油加工能力约1600万吨。
