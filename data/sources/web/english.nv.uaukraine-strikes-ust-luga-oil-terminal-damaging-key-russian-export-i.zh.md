# 乌克兰再度袭击俄罗斯能源设施 乌斯季卢加港又遭打击

2026年4月7日 16:14

![图片6：2026年4月7日无人机袭击后乌斯季卢加港起火画面（图片来源：乌克兰武装部队总参谋部）](https://static.nv.ua/shared/system/Article/posters/003/270/621/original/27d22ae83499a78b7ffd87bbbf5212f6.png?q=85&stamp=20260407181421&w=900&f=webp)

2026年4月7日无人机袭击后乌斯季卢加港起火画面（图片来源：乌克兰武装部队总参谋部）

乌克兰武装部队总参谋部表示，4月7日凌晨，乌克兰国防军袭击了俄罗斯列宁格勒州的乌斯季卢加石油终端。

总参谋部称，至少有三个属于Transneft-Baltika公司的储油罐被击中。官员补充说，该设施是俄罗斯石油出口基础设施的关键组成部分，其产生的收入被用于资助对乌克兰的战争。

总参谋部表示，在4月5日对Transneft-Port Primorsk设施的另一次袭击中，三个大容量储油罐受损，导致石油产品起火。每个容量为2万立方米的受影响储罐用于长期储存石油和其他易燃液体。

同日，对卢克石油-下诺夫哥罗德石油有机合成工厂的袭击损坏了关键的原油加工装置，包括AVT-6和AVT-1，以及用于沥青生产的19/6装置。

乌克兰Telegram频道Exilenova+报道称，在部分运营恢复后，乌斯季卢加港于4月7日再次遭到袭击。

列宁格勒州州长亚历山大·德罗兹坚科证实了无人机袭击事件，声称防空系统在该地区击落了22架无人机。

彭博社4月5日报道称，在中断数日后，该港已开始以最低水平恢复石油出口。

乌克兰安全局此前证实，远程无人机在3月下旬袭击了乌斯季卢加港的基础设施，卫星图像显示乌斯季卢加港和普里莫尔斯克港都发生了大火。

路透社报道称，3月31日无人机再次袭击列宁格勒州，这是10天内乌斯季卢加港第五次遭袭。

彭博社报道称，在一系列袭击事件后，俄罗斯波罗的海港口的石油运输量已降至2022年俄罗斯全面入侵乌克兰以来的最低水平。
