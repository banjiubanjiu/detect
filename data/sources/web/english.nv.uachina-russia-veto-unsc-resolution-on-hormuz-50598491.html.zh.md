# 中俄否决联合国安理会关于霍尔木兹海峡的决议

4月7日 20:27

国际

![图片4：联合国安理会（照片：路透社/Amr Alfiky）](https://static.nv.ua/shared/system/Article/posters/003/270/717/original/3996b2f435427ee31c24683ebaab50a8.png?q=85&stamp=20260407212522&w=900&f=webp)

联合国安理会（照片：路透社/Amr Alfiky）

据彭博社4月7日报道，中国和俄罗斯否决了一项旨在恢复[霍尔木兹海峡](https://english.nv.ua/nation/us-strikes-military-targets-on-iran-s-kharg-island-50598448.html)航运的联合国安理会决议。

这项由巴林提出的决议获得15票中的11票支持，但被中俄两国行使否决权阻止。投票前，海湾国家经过多日谈判施压，要求恢复被伊朗封锁的该海峡自由通行权。

巴林草案呼吁各国"酌情协调防御行动"，以确保霍尔木兹海峡航行安全。

此次投票正值美国总统特朗普设定最后期限前数小时。特朗普曾表示伊朗必须在东部时间晚8点（基辅时间4月8日凌晨3点）前重新开放海峡，否则将威胁对伊朗民用基础设施实施空袭。

投票结束后，支持该决议的美国驻联合国大使迈克·沃尔茨立即向安理会表示，1979年推翻巴列维国王后，阿亚图拉们的第一步就是"扣押数十名美国人质"。

"现在他们劫持了霍尔木兹海峡，并试图借此绑架世界经济，"沃尔茨说，"所以同事们，这可能是他们的最后一步。我们拭目以待。"
