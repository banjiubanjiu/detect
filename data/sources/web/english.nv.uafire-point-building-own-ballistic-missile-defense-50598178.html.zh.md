# 火点公司计划2027年前推出更便宜的爱国者替代系统

2026年4月7日 上午08:28

![图片6: 2.b.blogspot.com](https://static.nv.ua/shared/system/Article/posters/003/270/031/original/b33f79c7aa70558e46a7fe6fe1b2b086.png?q=85&stamp=20260406235315&w=900&f=webp)

2.b.blogspot.com

乌克兰导弹与无人机生产商[火点公司](https://english.nv.ua/russian-war/ukraine-s-fire-point-to-launch-rocket-fuel-production-in-denmark-by-2026-50592425.html)联合创始人兼首席设计师丹尼斯·什蒂列尔曼4月6日向[路透社](https://www.reuters.com/business/aerospace-defense/ukraine-missile-maker-targets-game-changer-air-defence-system-by-2027-2026-04-06/)表示，计划在2027年前开发自主防空系统，将成为爱国者系统的低成本替代方案。

什蒂列尔曼指出，爱国者系统通常需要发射2-3枚拦截弹才能击落一枚来袭弹道导弹，每枚拦截弹造价高达数百万美元。火点公司的目标是将其拦截弹成本控制在100万美元以下。

"若能将其成本降至百万美元以下，这将成为防空解决方案的革命性突破，"他向路透社表示。

"我们计划在2027年底实现首次弹道导弹拦截。"

路透社报道称，火点公司正与欧洲企业洽谈新系统研发合作，但什蒂列尔曼拒绝透露具体企业名称。他表示对雷达、导弹制导和通信等自身经验不足的领域"深感兴趣"。

报道指出，由于波斯湾地区的战事消耗，爱国者导弹库存持续减少，而欧洲SAMP/T系统的产量相对有限。
