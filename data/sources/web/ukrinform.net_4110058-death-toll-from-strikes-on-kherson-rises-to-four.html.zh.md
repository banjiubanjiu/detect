# 赫尔松袭击事件死亡人数升至四人

乌克兰国家通讯社

俄罗斯对赫尔松市科拉贝尔尼区发动的袭击已造成四人死亡。

据乌克兰国家通讯社报道，赫尔松州军事行政长官亚历山大·普罗库丁在Telegram上发布了这一消息。

普罗库丁表示："俄罗斯对赫尔松市科拉贝尔尼区发动袭击造成的死亡人数已升至四人。据报道，另有一名男子在袭击中受致命伤。"

他指出，目前正在确认遇难者身份。

乌克兰国家通讯社此前报道，俄罗斯对赫尔松市科拉贝尔尼区居民楼的袭击已造成三人死亡、七人受伤。
