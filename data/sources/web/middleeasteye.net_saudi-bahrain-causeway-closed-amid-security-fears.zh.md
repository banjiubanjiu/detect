# 特朗普威胁摧毁伊朗基础设施后 沙特-巴林跨海大桥关闭

2026年4月7日 06:55 BST

![](/sites/default/files/styles/max_2600x2600/public/2026-04/000_A6XH2EW.jpg.jpg?itok=az6vTcg6)

连接沙特阿拉伯与巴林的法赫德国王跨海大桥管理机构宣布，由于担心遭受袭击，已预防性暂停大桥通行。

此举源于对伊朗可能袭击沙特东部省的担忧。这座25公里（15.5英里）长的跨海大桥是两国间唯一的陆路通道。

在美国总统唐纳德·特朗普威胁打击伊朗基础设施，而伊朗警告若民用设施遭袭将进行"毁灭性"报复后，大桥关闭标志着紧张局势升级。
