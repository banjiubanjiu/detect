# 如果中国攻击台湾

### 邦妮·S·格拉泽导言

近年来，关于台湾海峡发生冲突的可能性和可能结果的研究迅速增加。研究集中于与威慑、潜在冲突动态和可能的冲突结果相关的广泛问题。桌面演习已被用于识别美国、中华人民共和国（PRC）和台湾在能力方面的差距，以评估潜在的升级途径，并更好地理解战争结束策略。1 相对而言，较少关注的是台海冲突对中国自身可能产生的影响，以及这种影响可能如何影响习近平主席对使用武力攻击台湾的风险计算和决策。

习近平的风险计算对于理解北京是否以及在何种情况下可能对台湾采取侵略行动至关重要，因为任何此类决定都将对中国以及他个人产生深远的政治、经济和战略后果。习近平已将他的合法性与使中国走上到2049年实现民族复兴的“中国梦”的不可逆转的道路联系起来，而统一台湾被认为是实现这一目标的关键。然而，一场关于台湾的军事冲突将面临大规模的经济破坏、灾难性的军事损失、严重的社会动荡和毁灭性的制裁的风险，所有这些都可能使他的梦想变成一场噩梦，并削弱他的政治权威。因此，习近平的计算必须权衡使用武力实现统一的感知利益与潜在成本。

习近平的风险计算对于理解中国未来关于可能对台湾采取行动的决策至关重要，这需要比迄今为止进行更深入的调查。本报告旨在通过考察对台湾使用武力将如何影响中国在四个关键领域的表现来填补这一研究空白：其经济、其军事能力、其社会稳定和国际成本。每个问题集都在单独的论文中进行评估，尽管所有四个问题集都是相互关联的，这是报告结论中涉及的一个因素。洛根·赖特和查理·韦斯特评估了潜在的

对中国经济的影响。乔尔·伍思诺分析了对中国军队的可能影响。希娜·切斯纳特·格雷滕斯和杰克·里纳尔迪考虑了对社会稳定的潜在风险。扎克·库珀考察了可能的国际反应和成本。

中国对台湾使用武力可能存在多种情况。北京可能采取的潜在行动在强度和持续时间上可能有所不同。为了使四篇论文之间能够进行比较，作者们被赋予了两个基线情景，即有限冲突和重大战争。两者都在下面详细描述。在概述这些情景时，没有建立具体的触发因素。然而，它们对于确定国际反应至关重要，并随后对中国经济以及可能对社会稳定产生影响。尽管缺乏详细的触发因素是一个限制因素，但在任何危机中，多个行为者都会对引发冲突的原因得出自己的结论，因此各国对谁有过错的看法会有所不同。

鉴于经济、军事、政治和国际成本可能在一系列情景中不连续地变化，作者们也被允许评估介于小冲突和重大战争之间的案例。所有情景都假定发生在2026年至2030年之间。作者们评估的第一个情景是持续数周的小冲突。在这种情况下，在一系列中国和台湾军队之间致命的空海对抗之后，中国船只和飞机包围了台湾。然后，中国人民解放军（PLA）试图对台湾的主要港口进行“隔离”。美国进行干预，其军舰护送商业船只通过封锁区，没有发生任何事件。为了本情景的目的，假定数十名中国和台湾武装部队成员在空中和海上事件中丧生。没有外国人员伤亡。当双方同意缓和局势时，冲突结束。这导致了一场相对低强度和短时间的冲突。

作者们被要求审查的第二个情景是一场持续数月的重大冲突，最终以解放军的失败告终。这场冲突始于对台湾的两栖入侵，其中解放军最初的导弹袭击目标是台湾的军队和驻扎在日本和关岛的美军。中国军队在台湾登陆，但由于台湾和美国对穿越台湾海峡的船只和飞机的持续袭击，补给和后续部队受到阻碍。经过几个月的激烈战斗，中国军队在损失约10万人员后撤回大陆。台湾遭受约5万名军事人员伤亡和5万名平民伤亡。美国损失5000名军事人员和1000名平民，而日本损失1000名军事人员和500名平民。除此之外，没有发生重大的外国损失。当解放军从台湾本岛撤退但保留对金门和马祖岛的控制时，冲突结束。这些情景的目的不是断言这就是台湾冲突的结局。相反，其目的是帮助更详细地描述中国在失败的跨海峡行动中所付出的代价。一个重要的警告是，作者们被要求对中国在所考察的四个领域中的每一个领域的影响做出自己的评估。他们没有被要求设身处地地站在习近平（或他的继任者）的立场上。习近平和他的同事们可能会低估（或高估）中国在这些情景中将承担的成本。即使北京预计对台湾采取行动的成本很高，他们也可能不顾感知的风险而继续行动。如果感知的利益很高或被政治考虑所取代，国家领导人往往会做出忽视高成本的选择。习近平可能会得出结论，对他个人而言，不采取行动比采取他认为有必要展示决心的冒险行动的代价更高。例如，如果习近平认为台北采取的步骤旨在将其永久地与中国分离，特别是如果他判断华盛顿支持这一目标，那么即使他预计解放军在试图夺取台湾时会遭受重大伤亡，他也可能会对台湾采取行动。总的来说，这些文章的作者们表明，中国对台湾采取的失败的军事行动的代价将是巨大的。如上所述，这并不是说北京必然会被阻止首先发动冲突。但是，简单地假设中国会在台湾冲突中获胜将是一个错误。失败的两栖作战的历史很长，这些文章表明，中国发起的失败的冲突将对该国的经济、军事、社会稳定和国际地位产生严重的负面影响。

*本文所表达的观点仅代表作者的观点。德国马歇尔基金会作为一个机构不采取立场。*
