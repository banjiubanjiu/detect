# 局势进展伊朗警告美国将为违反停火协议"付出代价" 期待中国担任地区安全保障者

德黑兰驻华大使表态数小时前 特朗普宣布暂停轰炸两周以促成停火谈判

![伊朗政府支持者在德黑兰集会庆祝与美国及以色列达成两周停火协议，4月8日。照片：美联社](https://cdn.i-scmp.com/sites/default/files/styles/1020x680/public/d8/images/canvas/2026/04/08/1ac44dc8-c6c1-493e-95d5-f283eefd55ba_44823c68.jpg?itok=UwJlq-cA&v=1775632649)

伊朗驻华大使阿卜杜勒礼萨·拉赫马尼·法兹利周三在北京表示，希望中国能成为该地区和平的安全保障方之一。

但他同时警告，若美国再次背信弃义，伊朗将发起强力"反击"。

"我们希望各方能共同确保美国不会重启战端，期待联合国安理会、中俄等大国以及巴基斯坦、土耳其等调解国携手保障地区和平，"法兹利在中国首都的记者会上表示。

此番表态数小时前，美国总统特朗普宣布暂停对伊朗的轰炸行动两周，为停火谈判创造条件。

特朗普在宣布决定后接受法新社采访时称，相信中国在促使伊朗回到谈判桌方面发挥了关键作用。

法兹利表示，伊朗始终与友好国家特别是中国保持着密切对话与合作。

"我们希望战火能够停止，停火得以延续，同时也需要可靠的安全保障。"
