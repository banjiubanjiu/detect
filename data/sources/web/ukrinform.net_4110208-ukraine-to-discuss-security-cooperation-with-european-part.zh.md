# 乌克兰将在未来数周与欧洲伙伴讨论安全合作事宜——泽连斯基

据乌克兰国家通讯社报道，泽连斯基在视频讲话中作出上述表态。

"我们正在准备安全提案的欧洲部分内容，这些提案已在最近几周中东和海湾国家的谈判中提出并获得认可。这些国家目前正处于战争状态——很不幸。他们的防卫需求需要紧急的专业支持和快速响应。我们确保了这一点——乌克兰的专业能力被证明是有效的——这一点获得了各方认可。最终我们达成了长期合作协议，特别是安全领域的合作。欧洲同样需要这样的合作，"泽连斯基表示。

他补充说，未来数周将重点推进与欧洲伙伴的此类合作：联合生产、国防现代化、国防融资计划以及安全行动协调。

乌克兰国家通讯社此前报道，泽连斯基还表示乌克兰正在准备关于加强安全保障文件的提案。
