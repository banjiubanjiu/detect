# 伊朗"不忘朋友"：马来西亚船只获准通过霍尔木兹海峡 凸显选择性准入政策

分析人士指出，此举标志着德黑兰将霍尔木兹海峡通行权作为政治筹码，同时引发对马来西亚石油转运活动的关注

分析人士表示，伊朗允许马来西亚关联船只通过霍尔木兹海峡的决定，凸显德黑兰正日益将这一战略水道的通行权作为政治筹码。目前仅有少数船只能够通过，准入资格越来越取决于政治关系，而非作为中立的商业权利。

对马来西亚而言，此举不仅暴露了德黑兰与布城的密切关系，也使其水域内敏感的石油贸易和船对船转运活动受到审查。

在总理安瓦尔·易卜拉欣直接向伊朗总统马苏德·佩泽什基安提出交涉后，至少一艘前往柔佛的马来西亚关联油轮于周日恢复航行。此举帮助七艘滞留在海峡内或附近的船只获得了通行许可。

伊朗驻吉隆坡大使馆在社交媒体上明确传达了政治信号，称首艘马来西亚船只通过海峡后，伊斯兰共和国"不会忘记朋友"。霍尔木兹海峡承担着全球约20%的石油运输，自美国-以色列对伊朗开战以来，该海峡的通行受到严格限制。

分析人士认为，伊朗似乎正在海峡推行更具选择性的准入模式，将通行权作为一种政治杠杆。
