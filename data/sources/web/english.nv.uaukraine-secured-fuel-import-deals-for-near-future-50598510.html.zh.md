# 泽连斯基：乌克兰已确保近期燃料进口协议

2026年4月7日 20:06

![泽连斯基官方/电报](https://static.nv.ua/shared/system/Article/posters/003/270/759/original/0be7614681b3c49f2ee83caf8947d47e.png?q=85&stamp=20260407231946&w=900&f=webp)

泽连斯基官方/电报

乌克兰总统弗拉基米尔·泽连斯基4月7日表示，乌克兰已与合作伙伴达成协议，将进口[所需燃料量](https://english.nv.ua/nation/politico-global-energy-crisis-strains-fuel-reserves-for-ukraine-s-military-50595360.html)至少维持到本月底。

"重要的是我们已达成向乌克兰供应燃料的协议，目前正在为三月和四月进口必要数量，"总统说。

"我感谢所有参与合同的企业——无论是国有还是私营企业。"

他表示乌克兰正与欧洲伙伴和中东国家安排燃料运输，称这是乌克兰国际合作新阶段的一部分。

泽连斯基还表示，政府必须制定措施在能源价格上涨背景下帮助公民。"我期待系统性解决方案，"他声明道。

由于伊朗战争及伊朗封锁霍尔木兹海峡后波斯湾国家石油供应中断，乌克兰燃料价格出现上涨。作为应对，内阁批准了燃料现金返还计划，允许司机获得部分燃油费用补偿。该计划自3月20日开始实施，将持续至5月1日。
