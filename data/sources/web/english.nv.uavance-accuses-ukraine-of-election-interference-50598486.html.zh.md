# 万斯指控乌克兰干预美国大选

2026年4月7日 下午04:57

![美国副总统JD万斯（照片：路透社/Jonathan Ernst）](https://static.nv.ua/shared/system/Article/posters/003/270/706/original/7f937b45228ac1b2923869dd0de75dba.png?q=85&stamp=20260407205623&w=900&f=webp)

美国副总统[JD万斯](https://english.nv.ua/nation/vance-visits-hungary-ahead-of-election-signals-support-for-orban-50598301.html)声称乌克兰政府体系中的某些"人员"曾试图干预2024年美国总统选举。

他在4月7日为匈牙利总理欧尔班助选时发表上述言论，此时距离可能终结欧尔班长期执政的高风险议会选举仅剩五天。[竞选视频](https://www.youtube.com/watch?v=iU0SEPSwxQA)

万斯表示，乌克兰安全部门的官员"试图插手"美国和匈牙利选举，并称这种干预是"与乌克兰体系中某些势力合作的代价"。"他们一贯如此，"他补充道。

他指控乌克兰体系内人员在2024年11月总统大选前"与民主党人开展合作"，《卫报》[报道](https://www.theguardian.com/world/2026/apr/07/jd-vance-eu-interference-hungary-election-viktor-orban)指出，这番言论是他在匈牙利助选时发表的。

万斯还表示，他认为所有人都希望俄乌战争尽快结束。

"无论乌克兰体系中某些势力对我或其他人有何看法，我坚信这场战争尽快结束符合乌克兰、欧洲、匈牙利和美国的利益，"他说。
