# 巴勒斯坦1936：一部关于几乎改写历史的起义的影片

导演安玛丽·雅西尔讲述《巴勒斯坦1936》如何将当今危机溯源至英国殖民统治。

在以色列占领之前，是英国殖民主义。我们与导演安玛丽·雅西尔对话，探讨这部关于1936年巴勒斯坦起义的史诗电影——这场几乎成功的抗争、当今危机常被遗忘的根源，以及为何这段历史仍令人痛切地感同身受。
