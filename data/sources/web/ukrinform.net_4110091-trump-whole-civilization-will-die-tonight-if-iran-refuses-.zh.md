# 特朗普警告：若伊朗拒绝协议"今夜整个文明将消亡"

据乌通社报道，特朗普在社交媒体平台Truth Social上发文称。

"整个文明将在今夜消亡，永远无法挽回。我不希望这种情况发生，但它很可能会。不过既然我们已经实现了彻底完全的政权更迭，由不同、更聪明且不那么激进的思想主导，或许会发生革命性的美妙转变，谁知道呢？我们将在今夜见证，这是世界漫长复杂历史中最重要的时刻之一，"特朗普声明道。
