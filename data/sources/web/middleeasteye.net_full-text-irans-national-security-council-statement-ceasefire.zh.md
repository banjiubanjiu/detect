# 伊朗最高国家安全委员会关于停火的声明全文

![2026年4月8日，德黑兰革命广场上民众对停火公告的反应。STR/AFP](/sites/default/files/styles/max_2600x2600/public/images-story/Iranians%20react%20after%20a%20ceasefire%20announcement%20at%20the%20Enqelab%20square%2C%20in%20Tehran%2C%20on%20April%208%202026..jpg.jpg?itok=Me3-Rbcx)

在美国-以色列对伊朗发动袭击并引发伊朗在全区实施导弹和无人机报复性打击后，持续数周的冲突升级终于迎来停火，为外交斡旋打开了一扇狭窄的窗口。

这场冲突导致全球能源要道霍尔木兹海峡航运中断，并加剧了地区大战的担忧。经巴基斯坦斡旋，美国总统唐纳德·特朗普宣布为期两周的停火，条件包括重新开放海峡以及在伊斯兰堡启动谈判。

双方均宣称取得军事成果——华盛顿表示已实现核心目标，德黑兰则坚称迫使美国接受了其要求框架。即将展开的谈判将聚焦海上通行、制裁解除及美军地区存在等关键议题。

以下是伊朗最高国家安全委员会停火声明全文：

"敌人在对伊朗民族发动的怯懦、非法与罪恶战争中，已遭受不可否认的历史性惨败。承蒙伊斯兰革命殉道领袖——尊贵的伊玛目哈梅内伊阁下（愿主赐他平安）——纯洁圣血的庇佑，以及伊斯兰革命最高领袖兼武装部队总司令——尊贵的赛义德·莫杰塔巴·哈梅内伊阁下（愿主护佑）——的睿智决策，加之伊斯兰战士们在各条战线——特别是自战争爆发首日起就英勇现身的人民群众——的奋战，伊朗已取得重大胜利，迫使罪恶的美国接受我方十点方案。

该方案中，美国必须从根本上承诺：保证不再侵略、承认伊朗对霍尔木兹海峡的控制权、接受铀浓缩权利、解除所有初级与次级制裁、终止安理会及理事会全部决议、赔偿伊朗损失、撤出地区作战部队、停止包括针对黎巴嫩英勇伊斯兰抵抗阵线在内的所有战线军事行动。

我们向全体伊朗人民祝贺这一胜利，并强调在最终落实胜利细节前，仍需官员们保持坚定睿智，维护国民团结。伊斯兰伊朗与黎巴嫩、伊拉克、也门及被占领巴勒斯坦的抵抗战士们在过去40天给予敌人的打击，将被全球历史记忆永久铭记。作为人类尊严代表对抗最野蛮敌人的伊朗与抵抗轴心，通过这场历史性战役给他们上了难忘的一课。

我们如此彻底地摧毁了敌人的部队、设施、基础设施及所有政治、经济、科技与军事资产，以致敌人现已陷入崩溃绝望，除屈服于伟大伊朗民族与崇高抵抗轴心的意志外无路可走。

战争首日，当伊朗的罪恶敌人发动这场不义之战时，他们妄图短期内取得对伊朗的完全军事支配，并通过制造政治社会动荡迫使伊朗屈服。他们以为伊朗的导弹与无人机打击将迅速熄灭，更不相信伊朗能跨境实施如此强有力的全域反击。

全球邪恶犹太复国主义势力蛊惑无知的美国总统，妄称此战将终结伊朗，使他们能在拔除这最后的人道堡垒后肆意妄为。他们梦想瓜分亲爱的伊朗、掠夺石油财富，最终让伊朗人长期陷入混乱动荡。

伊斯兰英勇战士们与抵抗轴心的勇敢盟友们在伊玛目殉道的悲痛中，仰仗全能真主并追随殉道者之主的指引，决心给这些敌人一个永世难忘的历史教训。他们选择清算所有历史罪行，创造让敌人永远放弃侵略伊朗念头的条件，使其在伟大伊朗民族面前饱尝屈辱滋味。

凭借这一战略及国内空前的政治社会团结，伊朗与抵抗阵线对美以政权发动了史上最猛烈的混合战争，在此阶段实现了所有既定目标。我们几乎完全摧毁了美国在地区的战争机器，对敌人多年经营的基础设施给予毁灭性打击，使罪恶美军承受重大伤亡，在被占领土上重创敌军力量与资产。

我们在各条战线形成压制态势，不仅使敌人主要目标全部落空，更令其在开战十天后就意识到绝无胜算。正因如此，敌人开始尝试通过多种渠道与伊朗接触乞求停火。

尊贵的伊朗国民须知：正是由于你们子弟的奋战与历史性现身，敌人乞求停火已逾一月。但我国官员自始便决定战争将持续至实现所有目标——包括使敌人悔恨绝望并消除国家长期威胁——故拒绝所有停火请求，直至第四十个战斗日的今天。

此外，伊朗迄今已拒绝美方提出的多个最后期限，并持续强调绝不重视敌人的任何时限威胁。现在我们向伟大伊朗民族报喜：几乎所有战争目标均已实现，你们的英勇子弟已迫使敌人陷入历史性无助与永久失败。基于全民一致支持，伊朗的历史性决定是将这场战斗持续至必要时刻，直到巩固重大成果并在地区建立承认伊朗与抵抗轴心力量主权的新安全政治秩序。

鉴此，根据伊斯兰革命最高领袖——尊贵的赛义德·莫杰塔巴·哈梅内伊阁下（愿主护佑）——的睿智决策与最高国家安全委员会批准，考虑到伊朗与抵抗阵线的战场优势、敌人无力实施威胁的事实及美方对伊朗人民正当要求的正式接受，决定在伊斯兰堡举行谈判敲定细节。这将确保在最多15天内，通过政治谈判巩固伊朗的战场胜利。

为此，伊朗在拒绝敌方所有方案后起草十点计划，经巴基斯坦转交美方。该方案强调：与伊朗武装部队协调下的霍尔木兹海峡管制通行（赋予伊朗独特地缘经济地位）、必须停止针对抵抗轴心所有成员的战争（标志杀童的以色列政权侵略的历史性失败）、美军撤出地区所有基地与部署点、建立保障伊朗主导权的海峡安全通行协议、全额支付伊朗估算损失、解除所有制裁与安理会决议、解冻海外资产，最终通过具有约束力的安理会决议批准所有条款。该决议将使这些协议成为国际法，为伊朗民族创造重要外交胜利。

尊敬的巴基斯坦总理已告知伊朗：尽管表面威胁不断，美方已接受这些原则作为谈判基础，并向伊朗民族意志屈服。据此最高层决定，伊朗将在伊斯兰堡与美方进行为期两周的谈判，且仅基于这些原则。必须强调这不意味战争结束，唯有在十点计划原则获接受且细节敲定后，伊朗才会接受终止战争。

谈判将于4月10日（波斯历法尔瓦尔丁月21日）周五在伊斯兰堡启动，伊朗对美方持完全不信任态度，并为谈判预留两周时间（可经双方同意延长）。此期间必须保持全民团结，继续强力开展胜利庆典。

当前谈判是国家谈判与战场的延续，所有民众、精英与政治团体必须信任并支持革命领袖与最高体系监督的这一进程，严格避免分裂性言论。若敌人在战场上的投降能转化为谈判中的决定性政治成果，我们将共同庆祝这一重大历史胜利；否则我们将继续并肩作战，直至伊朗民族所有要求得到满足。我们的手指仍扣在扳机上，敌人任何微小失误都将招致全力回击。"
