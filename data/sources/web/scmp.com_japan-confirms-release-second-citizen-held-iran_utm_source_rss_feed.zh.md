# 日本确认第二名在伊朗被扣公民获释

据信该人士为NHK德黑兰分社社长，系两名在伊朗被拘的日本公民之一

![据信NHK德黑兰分社社长川岛信之介（音）系周一获保释人员。照片：资料图片](https://cdn.i-scmp.com/sites/default/files/styles/1020x680/public/d8/images/canvas/2026/04/07/537b0dbb-ac1f-4aa5-b5fd-fc8642716621_2fc78884.jpg?itok=UYj1HgHA&v=1775530938)

东京方面周二证实，伊朗已释放自1月起被扣押的一名日本公民。但共同社报道称，该人士仍被禁止离开伊斯兰共和国。

这家日本通讯社援引东京政府匿名消息源报道称，据信获释者为日本放送协会（NHK）德黑兰分社社长。

政府发言人木原稔表示："日本驻伊朗大使馆已确认，1月20日被伊朗当局拘留的一名日本公民于当地时间4月6日获释。"

木原向记者表示："驻伊朗大使在获释后直接会见了该人士，确认其健康状况良好。"
