# 叙利亚：区域动荡中的局势

美国、以色列和伊朗之间的战争仍在继续，对包括黎巴嫩、伊拉克、卡塔尔、沙特阿拉伯、科威特、阿联酋和巴林等国在内的该地区构成了重大挑战。虽然叙利亚避免了军事上卷入当前的升级，但经济停滞、宗派两极分化加剧以及可能卷入黎巴嫩冲突的风险依然很大。

## 经济溢出效应与叙利亚脆弱的复苏

伊朗成功地扩大了冲突的地理范围——甚至通过扰乱霍尔木兹海峡的能源流动在全球范围内——引发了海湾地区几十年来最严重的经济危机之一[i]。这也将直接影响叙利亚遭受破坏的经济。尽管叙利亚没有驶往该海峡的船只，也没有可以成为目标的能源生产基地[ii]，但海湾国家遭受的经济损失将直接影响其承诺的重建计划。世界银行估计，饱受13年战争蹂躏的叙利亚经济需要至少2160亿美元才能重建[iii]。海湾国家，特别是卡塔尔和沙特阿拉伯，是振兴叙利亚经济计划的关键利益相关者，但他们的重点现在将首先放在重建本国经济和基础设施上。

展望未来几年，如果世界继续受到能源战争、通货膨胀和地缘政治不确定性的影响，叙利亚的经济复苏选择仍然有限。长期的区域不稳定可能会进一步推迟重建。鉴于估计有90%的叙利亚人生活在贫困线以下，整个社区和城镇沦为废墟，以及全国范围内日益严重的经济困难，叙利亚人已经走上街头。

## 以色列、真主党与叙利亚卷入的风险

叙利亚的邻国黎巴嫩正在经历伊朗战争最猛烈的冲击。在对伊朗开战不到48小时后，伊朗支持的真主党武装分子向以色列发射了六枚火箭弹，使黎巴嫩卷入了区域冲突升级。以色列的反应迅速而升级，对黎巴嫩各地进行了猛烈轰炸，造成1000多人死亡[iv]，近100万人流离失所[v]。以色列表示，如果真主党继续持有武器，它愿意采取与加沙地带类似的广泛行动[vi]。与此同时，黎巴嫩国家在体制上已经瘫痪，国民军仍然容易受到内部分裂的影响。

在这种背景下，有关叙利亚将参与解除真主党武装的传言甚嚣尘上，特别是在大马士革沿叙利亚与黎巴嫩边境集结军队[vii]之后。然而，叙利亚政府强调，这些是防御性安全措施，否认有任何加入以色列和真主党之间战斗的意图。同样，美国叙利亚问题特使汤姆·巴拉克重申[viii]，华盛顿没有要求大马士革在黎巴嫩“采取行动”。尽管有这些官方保证，但仍不能排除叙利亚无意中升级或间接卷入黎巴嫩冲突的风险。

大马士革现任政府与真主党有着长期的敌对历史，这不仅源于意识形态上的分歧，还源于大多数叙利亚人与真主党战士相关的暴力集体记忆。在叙利亚长达13年的冲突中，真主党在地面上发挥了至关重要的作用，并对叙利亚反叛分子和平民进行了大屠杀。尽管如此，自去年上任以来，叙利亚总统艾哈迈德·沙拉一直避免与真主党发生任何对抗。2025年8月，沙拉告诉[ix]一群黎巴嫩记者，他必须“放下我们因真主党在叙利亚的侵略而造成的伤口”。沙拉一直专注于在叙利亚境内建立政权，并构建一种新的外交政策姿态，以使该国免受外部冲突的影响。他似乎不太关心叙利亚境外的敌人，而更关心国内的挑战，同时牢记他的军队既没有能力也没有经验在本土以外作战。

美国可能对此有不同的看法，因为黎巴嫩缺乏独自解除真主党武装的后勤和政治能力。这可能对叙利亚和黎巴嫩来说都是灾难性的。

与1976年黎巴嫩的经历进行比较，突显了过去外部对叙利亚干预黎巴嫩的批准如何塑造了区域动态。50年前，在获得美国和以色列的许可后，哈菲兹·阿萨德向该国部署了叙利亚军队，以帮助马龙派对抗巴勒斯坦解放组织（巴解组织）。

阿萨德玩弄一方对抗另一方，以确保没有人获得足够的权力来挑战他自己的议程。这次行动给黎巴嫩的主权带来了长期后果：叙利亚军队在该国驻扎了数十年，控制了政治生活的方方面面。真主党进一步获得权力，而黎巴嫩国家则被削弱。

虽然今天的地缘政治条件不同，但叙利亚对黎巴嫩的任何干预都将产生更严重的后果。与1976年不同，叙利亚的干预很可能会被定义为宗派性质，并可能加剧现有的紧张局势，进一步破坏黎巴嫩已经脆弱的政治体系，并使黎巴嫩军队沿宗派界限分裂。

尽管大马士革政府似乎意识到了这些危险，但其完全控制地面事件的能力仍然不确定。叙利亚军队包括具有不同意识形态倾向的派别，最近发生的事件引发了人们对中央指挥的质疑。2025年3月沿海地区的冲突和7月在苏韦达的冲突表明，政府难以控制附属武装团体并防止地方暴力升级。

在黎巴嫩，宗派敏感性根深蒂固，对真主党在叙利亚所扮演角色的记忆仍然鲜明，风险可能会更加严重。

## 真主党和区域平衡的下一步是什么

目前伊朗冲突升级将如何结束尚不确定。然而，现在很明显，即使特朗普找到了摆脱战争的方法，以色列也决心继续在黎巴嫩的军事行动，其公开目标是消灭真主党。过去以色列在黎巴嫩的军事行动和特别行动（1992年、1996年、2006年和2024年）都采取了严厉报复和有针对性的暗杀政策，但这些行动并没有消灭真主党，这表明仅靠军事力量可能是不够的。事实上，以色列1982年的入侵以及随后对黎巴嫩南部的占领为真主党的出现创造了条件[x]。

今天，任何占领和扩大所谓的缓冲区都将证实真主党的合法性并增强其地位。另一方面，黎巴嫩国家的脆弱性将进一步团结什叶派社区，并扩大其支持基础，而不是削弱它。由于受到国家的忽视和黎巴嫩其他宗教社区的反对，支持者将继续将真主党视为他们的保护者。

对于叙利亚来说，强大的真主党意味着伊朗在其边境的强大存在，构成了持续的安全威胁。另一方面，真主党的混乱崩溃将导致跨界暴力宗派化和激进化，因为它将引发逊尼派-什叶派身份冲突。萨拉菲派民兵在叙利亚境内活动，政府似乎对他们几乎没有控制权。因此，与真主党战士的任何对抗都将进一步加剧叙利亚和黎巴嫩两极分化的局势，并可能将冲突定义为一场针对什叶派的全面战争。

至少目前对叙利亚来说，最好的情况可能是将真主党控制在一个强大的黎巴嫩国家内。为了实现这一目标，应将重点放在加强国家机构而不是个人领导人上。削弱真主党不能被视为短期目标，而且在没有黎巴嫩国家和以色列军队继续驻扎在该国的情况下，它永远不会发生。

大马士革和贝鲁特之间大约两小时的车程，启发了黎巴嫩和叙利亚的一句谚语，反映了这两个国家命运的交织程度：*当大马士革有乌云时，贝鲁特就会打开雨伞*，反之亦然。尽管美国和以色列对伊朗和中东地区平衡的最终目标仍不清楚，但叙利亚人和黎巴嫩人将在未来几十年内感受到其连锁反应。
