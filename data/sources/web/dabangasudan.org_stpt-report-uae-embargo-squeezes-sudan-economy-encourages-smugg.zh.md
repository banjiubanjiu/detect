# STPT报告："阿联酋禁运重创苏丹经济 助长走私活动"

![](https://www.dabangasudan.org/wp-content/uploads/2023/01/gold-5-1.jpg)

苏丹黄金交易（档案照片：RD）

**苏丹透明与政策追踪机构今日发布的报告警告称，阿联酋贸易限制与地区紧张局势正加剧苏丹经济危机，迫使更多交易转入非正规网络。**

报告指出，苏丹燃料供应已因更广泛的地区冲突而吃紧，警告称"任何路线封锁都将立即引发国内短缺"。

2025年8月，阿联酋实施海空禁运，封锁了通往苏丹港的关键路线。

报告称，这些措施"禁止船只使用阿联酋港口进行直接运输"，迫使贸易商转向更长、更昂贵的路线。

其结果是"进口成本"上升，而黄金和石油收入下降，加速了苏丹镑的崩溃。

报告补充道："增加的...运输时间和成本...推高了药品价格...以及进口食品成本"。

**"黄金贸易遭重创"**

禁运对苏丹主要出口部门打击最为严重。报告称，迪拜对苏丹黄金的关闭给"市场带来冲击波"，并指出阿联酋此前接收了近所有官方出口。

随着正规渠道萎缩，贸易已转向他处。报告警告称"它们转入...非正规和军事化网络"。

据报告，目前50%至80%的黄金产量被走私。

**"政府措施适得其反"**

据报道，喀土穆采取的集中黄金出口等应对措施使情况恶化。

报告称此类政策"适得其反地助长了走私"，重蹈历史覆辙。

燃料进口也受到冲击。报告指出苏丹"84.1%"的石油来自阿联酋，中断对民用和军事行动都造成严重影响。

**"平行市场"**

报告总结认为，禁运与政府管制都在强化战争经济。

"贸易限制...并未减少经济活动，而是将其导向非正规、军事化和跨境网络。"

报告警告称，若不实现贸易多元化和经济改革，苏丹仍将极度脆弱。
