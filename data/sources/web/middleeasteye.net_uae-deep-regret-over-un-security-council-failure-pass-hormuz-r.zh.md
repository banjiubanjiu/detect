# 阿联酋对联合国安理会未能通过霍尔木兹海峡决议表示"深切遗憾"

![](/sites/default/files/styles/max_2600x2600/public/2026-04/iran%20university%20bomb%20reuters.jpg.jpg?itok=YPsB7zM9)

阿联酋驻联合国代表团发表声明，对安理会未能"通过明确框架推动国际合作以终止伊朗非法袭击及对全球经济威胁"表示"深切遗憾"。

"霍尔木兹海峡必须对所有国家保持开放，航行自由必须得到保障。任何国家都无权切断全球贸易命脉，将世界推向经济灾难的边缘，"该代表团在X平台发布的声明中强调。

"安理会的不作为不会削弱这场危机的紧迫性，也不会动摇阿联酋的决心，"声明继续表示，并补充说阿联酋将继续推动国际社会确保海峡航行安全的努力。
