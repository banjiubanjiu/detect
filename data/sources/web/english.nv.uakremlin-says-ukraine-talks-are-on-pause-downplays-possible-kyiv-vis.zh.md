# 克里姆林宫拖延乌克兰谈判 淡化特朗普特使可能访问基辅

2026年4月6日 17:30

![图片6：俄罗斯独裁者弗拉基米尔·普京与克里姆林宫发言人德米特里·佩斯科夫在2025年10月10日塔吉克斯坦杜尚别举行的独联体峰会上。（照片：Sputnik/Grigory Sysoev/Pool via REUTERS）](https://static.nv.ua/shared/system/Article/posters/003/269/880/original/32c670854458111748cf632fcc4a9636.png?q=85&stamp=20260406174933&w=900&f=webp)

俄罗斯独裁者弗拉基米尔·普京与克里姆林宫发言人德米特里·佩斯科夫在2025年10月10日塔吉克斯坦杜尚别举行的独联体峰会上。（照片：Sputnik/Grigory Sysoev/Pool via REUTERS）

作者：[Alex Stezhensky](https://english.nv.ua/korrespondents/alex-stezhensky.html)

克里姆林宫发言人德米特里·佩斯科夫表示，由于目前难以以此形式召集会议，涉及乌克兰、俄罗斯和美国的三方谈判已暂停。俄罗斯宣传机构国际文传电讯社4月6日报道了这一消息。

佩斯科夫称谈判进程"目前处于暂停状态"，因为"美国人还有许多其他事务要处理"。

"因此现在很难举行三方会谈，"他表示。

当被问及美国代表团可能访问基辅时，佩斯科夫称克里姆林宫没有确凿信息表明正在计划此类访问。

乌克兰总统办公室主任基里洛·布达诺夫在4月4日发表的彭博社采访中表示，乌克兰预计由美国总统唐纳德·特朗普特使史蒂夫·维特科夫和特朗普女婿贾里德·库什纳率领的美国代表团将于4月到访。

据布达诺夫称，该代表团可能在今年4月12日东正教复活节后不久抵达。他还表示共和党参议员林赛·格雷厄姆可能加入该团队。

4月3日，布达诺夫在CEO Club Ukraine会议上表示，乌克兰、俄罗斯和美国参与的和平谈判进程仍在继续，但"极其困难"。

4月1日，乌克兰总统弗拉基米尔·泽连斯基通过视频连线参加了与美国官员的会谈。乌克兰方面还有国家安全与国防委员会秘书鲁斯捷姆·乌梅罗夫出席，美方代表包括特朗普特使史蒂夫·维特科夫和特朗普女婿贾里德·库什纳。
