# 2022年以来已记录600余起针对征兵官员的袭击事件

2026年4月7日 下午2:28

哈尔科夫州记录的征兵官员遇袭事件数量最多（照片：切尔诺夫策州地区征兵中心通过Facebook发布）

乌克兰国家警察4月7日向国际文传电讯社[透露](https://interfax.com.ua/news/general/1157326.html)，自2022年2月24日俄罗斯全面入侵以来，当局已记录611起针对履行公务的地区征兵中心官员的袭击事件。

已有三名征兵官员在此类袭击中丧生。案件数量最多的地区是哈尔科夫州（68起）、基辅市（53起）和第聂伯罗彼得罗夫斯克州（44起）。

警方指出案件数量呈逐年上升趋势：2022年5起，2023年38起，2024年118起，2025年341起。2026年至今已记录超过100起抗法事件。

仅过去一周就发生了三起严重袭击。

4月2日，利沃夫市一名52岁征兵中心军人颈部被刺身亡。

国防部长米哈伊洛·费多罗夫二月表示，全面征兵制度改革工作正在进行中。
