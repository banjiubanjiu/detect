# 乌克兰安全局称俄军总参情报局(GRU)通过被黑路由器实施间谍活动

2026年4月7日 19:27

![图片6：Sora Shimazaki/Pexels](https://static.nv.ua/shared/system/Article/posters/003/270/754/original/11784d5e3b382264cb3a76fa6786ed36.png?q=85&stamp=20260407230005&w=900&f=webp)

Sora Shimazaki/Pexels

乌克兰安全局(SBU)4月7日宣布，在与美国联邦调查局(FBI)及欧盟执法机构合作下，成功挫败了俄罗斯军事情报总局([GRU](https://english.nv.ua/nation/russian-spy-from-secret-gru-unit-detained-in-colombia-after-google-translate-exposed-plot-50591733.html))利用被入侵Wi-Fi路由器对欧美及乌克兰公民进行监控的行动。

这项跨国网络行动发现，GRU特工通过存在安全漏洞的老旧办公及家用Wi-Fi路由器实施入侵。获取控制权后，攻击者将被黑路由器的流量重定向至恶意DNS服务器网络。这些服务器使攻击者能够截获密码、身份验证令牌等敏感信息，包括通常受加密保护的电子邮件内容。

SBU指出，GRU特别针对政府公务员、军官及国防工业从业人员展开监控。

此次行动查封了100多个恶意DNS服务器，并将乌克兰境内数百台路由器从俄方控制中解放，显著削弱了GRU的情报收集能力。

SBU敦促路由器用户及时更新固件并安装安全补丁。若厂商已停止技术支持，建议更换新型号设备，包括改用其他品牌产品。
