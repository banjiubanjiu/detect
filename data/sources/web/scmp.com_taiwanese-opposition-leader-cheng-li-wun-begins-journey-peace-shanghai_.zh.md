# 台湾在野党领袖郑丽文开启上海"和平之旅"

台湾在野党领袖郑丽文周二启程前往上海时表示，她希望让台湾海峡变得"安全"，不再是世界上最"危险的地方"之一。

国民党主席郑丽文正率领一个14人代表团——包括三位国民党副主席——展开为期六天的大陆访问行程。

此行计划与中共领导人习近平会面的安排，在台海局势紧张加剧之际引发台北方面高度关注。

广告

执政的民进党立法委员对会谈内容表示担忧，并警告国民党与北京的交流无法受到监督。

行政院长卓荣泰呼吁加强对与共产党往来的政治影响力人物的审查。
