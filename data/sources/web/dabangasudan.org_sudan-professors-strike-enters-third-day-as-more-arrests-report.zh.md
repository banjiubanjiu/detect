# 苏丹大学教授罢工进入第三天 更多逮捕事件被报道

![杰济拉大学教师参与全国罢工行动-2026年3月30日-社交媒体](https://www.dabangasudan.org/wp-content/uploads/2026/03/إضراب-أساتذة-جامعة-الجزيرة.jpg)

2026年3月30日，杰济拉大学教授参与苏丹大学教授委员会发起的全国罢工行动（图片来源：社交媒体）

**苏丹大学教授的全国无限期罢工昨日进入第三天，组织者称参与率达到百分之百，同时安全部队被报道逮捕学者并驱散抗议活动。**

苏丹大学教授委员会表示，周一罢工响应率已达100%，高于首日的95%，不过达班加电台无法独立核实这一数据。

委员会报告称，欧拜伊德安全部队逮捕了科尔多凡大学的11名教授，罪名是参与罢工并组织静坐抗议。

据委员会透露，全国35所大学的数万名教授在校园内和主要入口处协调举行静坐抗议，高举列明诉求的横幅。

**"蓬勃发展的工会运动"**

此次罢工标志着苏丹工会活动的重大升级，教授们要求实施修订后的薪资结构和改善服务条件。

包括教师和工会组织在内的专业团体已声援罢工行动，同时高等教育机构的工作人员据称正在筹备进一步的行业行动。

一个教师委员会也组建了高级罢工机构，作为其描述的关于工资、欠薪和工作条件的升级运动的一部分。

工会观察人士表示，罢工浪潮反映了在物价上涨和通货膨胀背景下工人面临的经济压力日益加剧，可能预示着苏丹有组织劳工活动的广泛复苏。

达班加电台无法独立核实逮捕人数或参与率。

**"逮捕与抗议驱散"**

达班加电台昨日报道了苏丹大学教授委员会强调的对学者的镇压行动。据报道，青尼罗州首府埃德达马津的安全部队因罢工逮捕了巴赫特·里达大学的教授，其中大多数人后来获释，至少两人仍被拘留。

在恩图曼的圣训与伊斯兰科学大学，警方也驱散了静坐抗议活动并短暂拘留了两名学者。委员会谴责逮捕行为并要求释放仍被拘留者。
