# 习特峰会前仅举行虚拟会谈且无新投资推动：格里尔

美国贸易代表强调美方寻求实质性关税、无阻碍稀土供应及维稳性质的元首会晤

![美国贸易代表杰米森·格里尔在白宫外向记者发表讲话。照片：美联社](https://cdn.i-scmp.com/sites/default/files/styles/1020x680/public/d8/images/canvas/2026/04/08/0247718e-40e6-4fbc-a520-7ef0da5eb772_7f0d9039.jpg?itok=ePkWNXZn&v=1775599096)

美国贸易代表杰米森·格里尔周二表示，与中方筹备会议将以虚拟形式进行，排除了峰会前亲赴北京的可能性，尽管工作层级的磋商仍在继续。

此次会议是在美财政部长贝森特与中国副总理何立峰率领的代表团进行新一轮会谈后举行的。

上周格里尔称，巴黎谈判人员已就五月峰会"成果类型达成总体共识"，并概述了组建"贸易委员会"来推动"管理型贸易"的计划——该体系基于双方互相采购等值商品的原则。
