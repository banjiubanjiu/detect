# 乌克兰根据IMF计划将战后军事税延长三年

2026年4月7日 15:55

![乌克兰最高拉达支持将战后军事征兵延长3年（照片：Martin Schulz/Flickr）](https://static.nv.ua/shared/system/Article/posters/003/270/633/original/4ebe1eaea0bacb60bc41c355b014a32d.png?q=85&stamp=20260407182440&w=900&f=webp)

乌克兰最高拉达支持将战后军事征兵延长3年（照片：Martin Schulz/Flickr）

据[国际文传电讯社4月7日报道](https://interfax.com.ua/news/economic/1157406.html)，乌克兰议会通过了将战后军事税延长三年的法案，这是国际货币基金组织新计划中的一项结构性基准，第15110号法案获得了257名议员支持。

财政部长谢尔希·马尔琴科在议会提交该法案时表示："法案的通过将使乌克兰国家预算在军事状态终止或解除后的三年内获得超过1400亿格里夫纳的收入。"

根据第15110号法律草案的说明，该法案规定延长在乌克兰军事状态期间实施的军事税征收条款。具体包括：个人缴纳5%；采用简化税制的第1、2、4类个体工商户（FOP）——按当月首日最低工资标准（2026年为850格里夫纳）的10%计算；第3类单一税纳税人（FOP和法人实体，不包括电子居民）——按收入的1%缴纳，期限为军事状态终止或解除后的三年。

议员雅罗斯拉夫·热列兹尼亚克表示："另一项与IMF相关的第15111号数字平台征税法案未列入议程。仅差一票未能通过。已推迟至明日审议。"

国际文传电讯社报道，乌克兰原定在2026年3月底前完成IMF扩展基金安排（EFF）计划中的四项结构性基准均未达标。议会本应通过相关税法修正案，包括取消简化税纳税人的增值税豁免、数字平台征税、所有包裹征税以及永久性军事税征收。

据报道，内阁于3月30日批准了财政部制定的三项税法修正案草案。

马尔琴科指出，拟议的税收改革包括规范数字平台收入征税、对零欧元起征的国际包裹征税，以及在军事状态结束后延长军事税征收。

马尔琴科还强调，关于部分个体工商户增值税的法案"目前正处于与中央行政机关协调修订阶段，将在近期提交审议"。
