# 美军空袭波斯湾伊朗哈尔克岛军事目标

2026年4月7日 15:14

![哈尔克岛石油码头，2026年2月25日（照片：Planet Labs PBC/Handout via REUTERS）](https://static.nv.ua/shared/system/Article/posters/003/270/624/original/f152671c87f67f837deeb1ffeee0c42e.png?q=85&stamp=20260407181541&w=900&f=webp)

哈尔克岛石油码头，2026年2月25日（照片：Planet Labs PBC/Handout via REUTERS）

据Axios记者巴拉克·拉维德4月7日在X平台援引美国官员消息，美国对波斯湾伊朗哈尔克岛军事目标实施了打击。

[根据阿纳多卢通讯社报道](https://www.aa.com.tr/en/middle-east/explosions-rock-iran-s-kharg-island-after-us-israeli-strikes-media/3896128)，伊朗媒体4月7日周二也报道称，在美以联合空袭后，哈尔克岛传出爆炸声。

此外，一名美国高级官员[告诉](https://x.com/JenGriffinFNC/status/2041479295667736673)福克斯新闻记者詹妮弗·格里芬，美国在夜间对哈尔克岛数十个军事设施实施了打击。

打击目标包括掩体、雷达站和弹药库。登陆码头并非预定目标。消息人士称，只有在伊朗部队在其附近开火时才会打击这些设施。

该官员补充说，此次行动完全由美国单独实施，以色列未参与。

"这是向伊朗人传递的信号，"这位美国高级官员表示。

据美国军方消息人士透露，如果伊朗铁路系统遭到打击，并非美军所为。

美国总统唐纳德·特朗普多次警告，如果德黑兰不达成协议或重新开放霍尔木兹海峡，美国可能打击伊朗的发电厂、桥梁等基础设施。

4月5日，他表示伊朗必须在东部时间周二晚8点前达成协议。

"周二将是伊朗的'发电厂日'和'桥梁日'，同时到来。这将非常壮观!!! 打开海峡，你们这些疯狂的混蛋，否则你们将生活在地狱里。等着瞧吧！赞美真主，"特朗普在Truth Social上写道。

德黑兰回应称，如果美国继续打击伊朗基础设施，"地狱之门"将向美国和以色列敞开。据CNN报道，德黑兰表示只有在获得战争损害赔偿后才会全面重开霍尔木兹海峡。

4月6日，Axios报道称美伊双方正通过中间人讨论为期45天的潜在停火条款，这可能最终结束战争。

同日，路透社援引消息人士称，伊朗拒绝了包括以重开霍尔木兹海峡换取停火在内的结束与美以战争的提议。

随后CNN报道称，特朗普不支持调解人提出的方案。

4月6日，特朗普表示伊朗可能在一夜之间被摧毁，并补充说最早可能在周二就会发生。
