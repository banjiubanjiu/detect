# 乌克兰经济安全局挫败国防企业采购舞弊案

据乌克兰国家通讯社报道，经济安全局新闻处公布了这一消息。

声明称："调查证实，该企业原物流采购部门负责人利用职务便利，与外国公司签订电容器供货合同，采购价格严重虚高于实际价值。"

该合同的签订缺乏合理经济依据，采购条款不符合国家资金高效合理使用原则。

这导致国有企业蒙受超过500万格里夫纳的经济损失。

涉案前官员已因触犯《乌克兰刑法典》第364条第2款（滥用职权造成严重后果）被正式立案起诉，相关起诉书已移送法院。
