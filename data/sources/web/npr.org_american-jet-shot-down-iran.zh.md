# 美方官员证实一架美军战机在伊朗境内坠毁

**切换说明文字**据一名未获授权公开表态的美方官员透露，一架美军战斗机在伊朗境内坠毁，其中一名机组人员已获救。该官员表示，美军部队正在搜寻第二名机组人员。

伊朗官方媒体最先报道了战机被击落的消息。

伊朗法尔斯通讯社发布在Telegram上的照片显示，疑似美军战机的残骸散落在地。其中一块残骸与F-15"攻击鹰"战斗机的垂直尾翼极为相似。另一张由伊斯兰革命卫队关联频道发布的照片中，可见一个空的F-15弹射座椅。

美国国家公共电台(NPR)暂无法核实这些照片的具体拍摄地点。

根据美国空军资料，F-15属战术战斗机，通常配备两名机组人员。

多段视频显示，伊朗西南部空域出现酷似美军搜救(SAR)飞机的固定翼飞机与直升机编队。NPR通过地理定位确认其中一段视频拍摄于胡齐斯坦省某桥梁附近。

该桥梁位于内陆约100英里处的山区。视频中可见一架飞机在低空飞行状态下为两架直升机实施空中加油。

视频显示疑似C-130运输机与两架"黑鹰"直升机在伊朗西南部胡齐斯坦省3 Balotak悬索桥附近执行搜救任务。

另一段视频疑似拍摄于胡齐斯坦省贝赫巴汉市附近，但具体位置尚不明确。

另据周五消息，一名美方官员向NPR透露，第二架单座战斗机在霍尔木兹海峡附近被击落，该名飞行员已获救。

今年3月中旬，一架美军F-35战机曾遭导弹击中，导致机体严重受损并造成飞行员受伤。

NPR当时获悉，这架F-35虽成功返回基地，但实施了硬着陆。飞行员被弹片所伤，接受了缝合治疗。
