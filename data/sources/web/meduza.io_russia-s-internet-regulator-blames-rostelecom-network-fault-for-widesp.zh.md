# 俄罗斯互联网监管机构将大规模断网归咎于Rostelecom网络故障

俄罗斯联邦通信、信息技术和大众传媒监督局（Roskomnadzor）向圣彼得堡媒体《丰坦卡报》表示，4月6日晚间俄罗斯境内银行网站、应用程序、电信运营商及其他服务的大范围瘫痪，是由Rostelecom骨干网络故障引起的。

前一日晚间，俄罗斯用户报告阿尔法银行、NTV Plus、Tricolor、国家服务门户（Gosuslugi）、T2、莫斯科能源销售公司（Mosenergosbyt）等服务的网站和应用程序出现故障。

用户还反映Rostelecom的家庭互联网服务在莫斯科中断运营。该公司声明称其网络遭到大规模分布式拒绝服务攻击（DDoS），并已迅速予以阻断。

4月7日上午，俄罗斯用户持续报告国家服务门户仍存在访问障碍。

近几周来，Roskomnadzor正积极封禁俄罗斯境内的Telegram服务。三月下旬，俄当局也加强了对VPN服务的打击力度。

在此背景下，四月初俄罗斯多个网站和应用程序接连发生大规模故障。4月3日，俄罗斯银行服务系统遭遇严重中断，导致客户数小时内无法转账、取现或刷卡支付。

专家认定事故责任方为Roskomnadzor。该机构要求媒体删除相关指控报道。

卡巴斯基联合创始人纳塔利娅·卡巴斯基曾表示Roskomnadzor在疯狂封禁翻墙工具时瘫痪了俄语互联网半数服务，随后被迫公开道歉。
