# SSU拘留协调克拉马托尔斯克空袭的俄罗斯特工

据乌克兰国家通讯社报道，乌克兰国家安全局（SSU）公布了这一消息。

经调查确认，该嫌疑人长期追踪乌克兰国防军人员及装备部署位置，俄军计划使用包括FAB-1500航空炸弹（配备滑翔模块）在内的多种武器对这些目标实施打击。

该特工的另一项任务是识别并向俄罗斯联邦安全局（FSB）传送存放乌军武器、弹药及补给物资的后勤仓库坐标。

SSU工作人员通过长期监控，完整记录了嫌疑人的间谍活动并将其逮捕。

![](https://static.ukrinform.com/photos/2026_04/1775467410-860.png)

案件材料显示，这项敌对任务由一名被FSB招募的当地无业人员执行。2025年10月，此人在Telegram聊天室中为获取快钱主动接触俄罗斯特工部门。

为收集情报，犯罪者通过实地踩点发现军事设施后，在谷歌地图上标记位置并将对应截图保存在智能手机中。

此外，该特工还在日常交谈中从其熟人处套取关于顿涅茨克地区防御部队的情报。

执法人员在对嫌疑人住所的搜查中，查获了存有其为敌方工作证据的移动设备。

SSU调查人员已依据《乌克兰刑法》第111条第2款（战时叛国罪）向其宣布嫌疑人身份。

犯罪者将面临终身监禁及财产没收。

乌克兰国家通讯社此前报道，一名向俄方传递基辅军事设施坐标的乌克兰公民被判处15年监禁。
