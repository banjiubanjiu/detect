# 撤离顿巴斯将使俄罗斯征服合法化 — 谢尔希·哈尔马什

2026年4月6日 下午7:35

![图6：路透社](https://static.nv.ua/shared/system/Article/posters/003/270/016/original/9fcb9b62ae32ded1eddeaf0522186550.png?q=85&stamp=20260406225156&w=900&f=webp)

路透社

乌克兰《岛屿》杂志主编谢尔希·哈尔马什4月6日向NV电台表示，如果乌克兰接受[俄罗斯要求](https://english.nv.ua/nation/kremlin-signals-no-quick-peace-denies-setting-ukraine-a-donbas-withdrawal-deadline-50597504.html)从顿涅茨克州撤军，将永远失去顿巴斯地区。

"如果我们自愿撤军，就意味着乌克兰在事实上和法律上承认该地区并入俄罗斯联邦的合法性，即承认这片领土属于俄罗斯。"哈尔马什说。

哈尔马什表示，虽然他从军方人员处听说乌军可能还能坚守顿巴斯约一年半时间，但他不确定具体能坚持多久。他警告说，按照莫斯科条件同意的自愿撤军是不可接受的，因为这将使俄罗斯的行动合法化，并关闭通过外交手段收复领土的可能性。

"这将意味着承认并合法化占领行为——不仅是占领，更是对该领土的吞并，承认其为俄罗斯领土。"他说，"这将永远关闭我们未来通过外交手段收回该领土的可能性。"

哈尔马什指出，俄罗斯独裁者弗拉基米尔·普京要求撤军的主要目的，就是要使占领行为合法化，并创造一个可适用于克里米亚及其他占领区的先例。

"这将成为任何谈判中的论据：'你们撤出了军队，就等于承认这是俄罗斯领土；你们无权要求归还。'"他说。

"这将彻底破坏国际公认边界和领土完整的原则。对我而言，这种行动的政治意义更为重要。我相信对俄罗斯来说，这才是主要目标——因为它将使其在乌克兰的行动永远合法化。"
