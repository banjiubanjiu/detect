# 乌克兰科学家在南极和世界海洋部署新型空气质量监测设备

据乌通社报道，乌克兰国家南极科学中心在脸书发布了这一消息。

公告称："南极通常拥有地球上最清洁的空气，污染水平极低。但监测来自其他大陆的污染物传输仍然至关重要。"

2024年，弗纳德斯基科考站安装了首台IQAir网络空气质量分析仪，可测量不同粒径的空气悬浮颗粒物浓度。

![](https://static.ukrinform.com/photos/2026_04/1775561868-666.jpg)

2026年，为校准首台设备，制造商向科考站提供了第二台相同型号的分析仪。

这将有助于避免测量中可能出现的技术误差，因为乌克兰科考站目前是南极地区唯一开展此类研究的场所。

![](https://static.ukrinform.com/photos/2026_04/1775561868-232.jpg)

设备由第31次考察队臭氧专家谢尔希·雅库申科和季节性气象学家奥列克桑德尔·纳托奇完成安装。

研究表明，科考站附近的污染水平通常接近于零，证实该地区空气确实属于全球最清洁之列。

同时，南极大气中的气溶胶含量会因自然因素（如火山爆发和风暴）、人为影响及气候条件（湿度和风力）而变化。

![](https://static.ukrinform.com/photos/2026_04/1775562187-308.jpg)

平流层中高浓度的气溶胶会引发破坏臭氧层的化学反应。

本季度，"Noosfera"号破冰船上也安装了空气质量分析仪。

此外还配备了AirGradient监测设备，可测量PM2.5颗粒污染物、二氧化碳、氮氧化物、挥发性有机物浓度，以及相对湿度和温度。

研究负责人亨纳迪·米林涅夫斯基表示："这些仪器用于监测海洋上空的气溶胶——而海洋正是永久监测站点最稀少的区域。"

"目前我们缺乏足够数据来建立影响气候与天气的气溶胶污染全球模型。因此在远洋船舶上开展气溶胶测量对扩充数据库至关重要。"科学家补充道。

国家南极科学中心向IQAir及其乌克兰官方合作伙伴Ecozen项目对研究的支持致谢。

得益于这项支持，乌克兰科学家将持续监测空气质量——这直接关系到地球上所有生命的生存质量。

据乌通社此前报道，乌克兰南极考察队在航行途中曾偶遇多对座头鲸。
