Methodology · 方法论

×
Escalation Index freq×0.5 + goldstein×0.3 + mentions×0.2
0-100 综合评分。比对近 7d 与前 7d 的事件量、平均 Goldstein 分数、外部提及次数。≥62 升级 / ≤38 缓和 / 中间稳定。
Goldstein Scale range -10 ~ +10
GDELT 用的事件烈度刻度。-10 = 最严重武装冲突,+10 = 最积极合作。我们仅采集 ≤ -5 的负面事件。
Cluster (多源印证) title Jaccard ≥ 0.5
同一冲突 ±2 天窗口内,标题 token 相似度达到阈值的事件被聚为一个 cluster。≥2 个独立源的 cluster 视作"印证"。GDELT 不参与(机器抽取不算独立源)。
Cross-bias (跨视角) cluster_bias_count ≥ 2
一个 cluster 的成员来自 ≥2 种媒体偏见标签 (如 ukrainian + russian-independent),视作"跨视角印证",可信度更高。
Criticality 分级 LLM 标注 (BLUF)
每条事件由 LLM 评为 critical / notable / background 三级。Critical = 必须知道,Notable = 值得关注,Background = 上下文。
SURGE 异常 7d > 1.5 × 前 7d 且 ≥ 10 条
某冲突近 7 天事件量超过前 7 天 50%,且绝对值达到 10 条以上,触发 SURGE 红标。无基线时要求 ≥ 15 条。
Bias Labels manually curated
每个媒体源(域名)预先标注: western / arab / russian-state / russian-indep / ukrainian / chinese / iranian / israeli / neutral。维护在 data/source_credibility.json。
Tier (可信度等级) T1 / T2 / T3
T1 = 学术/IGO/智库 (UN, ISW, CFR, HRW),T2 = 一线主流媒体 (NYT, BBC, Reuters, Al Jazeera),T3 = 区域/独立 (Meduza, Ukrinform)。
